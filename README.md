# Gourmet Battle Royale (Color Game & Search)

ホットペッパーグルメサーチAPIを使用した飲食店検索機能と、リアルタイム対戦型の色当てゲームを統合したWebアプリケーションです。

## 🚀 機能概要
* **飲食店検索:** エリアやジャンルを指定して飲食店を検索・一覧表示
* **Color Game:** ランダムな色に対する正解を素早く選ぶリアルタイムクイズ
* **CI/CD:** GitHub Actionsによる自動Lintチェック・ビルド検証

## 🛠 技術スタック
* **Frontend:** React, TypeScript, Vite
* **Styling:** Tailwind CSS
* **Backend(API):** Vercel Serverless Functions (Node.js)
* **API:** Hot Pepper Gourmet Search API
